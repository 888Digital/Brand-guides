The B2B Reseller Mindset

Key Insight: Your customers don't read marketing copy - they scan for financial opportunities. They think: "Can I make money? How much? How fast? What's the risk?"
The 3-Second Rule: A busy reseller should be able to spot profit potential, key dates, and brand names within 3 seconds. Everything else is secondary.

Non-Negotiable Readability Standards

✓ ALWAYS READABLE

  • • Prices: 18px+ bold, high contrast color
  • • Discount %: 16px+ bold, bright color
  • • Brand names: 16px+ bold, never truncated
  • • Quantities: 16px+ bold with clear units
  • • Critical dates: 16px+ bold when urgent
  • • Categories: 14px+ medium for filtering
  • • Mobile: All above sizes apply to mobile

✗ READABILITY KILLERS

  • • Small financial data (under 16px)
  • • Abbreviated brand names (Samsung → Sam)
  • • Unclear quantities ("bulk available")
  • • Hidden discount percentages
  • • Buried pricing information
  • • Low contrast text on backgrounds
  • • Critical info that requires zooming on mobile

Practical Application: Banners & Emails

Single-Column Email Mandate

Based on GA4 data showing high mobile user engagement, all marketing emails MUST use a single-column layout. This is non-negotiable.

Email Design Checklist

  • Single-column for perfect mobile rendering
  • Critical financial data & brand name "above the fold"
  • Clear, benefit-focused subject line (e.g., "70% Off Apple MacBook Lots")
  • Large, obvious Call-to-Action (CTA) button
  • Follows all readability font size rules
  • Unsubscribe link is clearly visible in the footer
